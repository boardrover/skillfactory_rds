## Что не получилось сделать так, как хотелось? Над чем ещё стоит поработать?

Пока не получается писать в "Pythonic-style", несмотря на попытку следования PEP-8. Катастрофически не хватает практики применения и времени для написания кода - это приводит к тому, что выработка навыков использования именно преимуществ Python идет медленней, чем хотелось бы.

Без этого следование PEP-8 оказывается синтетическим, а использование кратких форм записи для преобразования данных забывается и требуется заставлять себя откатываться назад, чтобы заставить использовать именно "питон"овские особенности.

Осталась загадкой функция reduce(), с этим еще предстоит разбираться подробней.

Дополнительно выяснил для себя, что обучающая платформа SkillFactory непригодна для справочной работы - очень медленно загружается, в связи с чем оказывается существенно быстрей найти ответ в поисковике/StackOverflow/проч., чем вернуться в прошлую главу и что-то освежить из пройденного.
